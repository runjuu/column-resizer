A library for multi-column(row) resizing. Works in every modern browser [which](https://caniuse.com/#feat=flexbox) supports [flexible box layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout).

- Support both mouse and touch events
- Support server-side rendering
- Support RTL (right-to-left) layout (Contributed by [@mike-shtil-loop](https://github.com/mike-shtil-loop))
- 0 dependencies, packages size under 5 kB after min-zipping
- Easy to customize resize behavior

- Or see [our examples](https://github.com/runjuu/column-resizer/tree/main/docs/examples) for inspiration.
